(static) DELETE Genre(:mediaID, :genreID, frdopt)

Flags a genre record as removed or deletes a genre record from the database based upon the frd parameter.

Parameters:
Name Type Attributes Default Description
:mediaID ObjectID Route Parameter. Specifies the ID of the media resource.
:genreID ObjectID Route Parameter. Specifies the ID of the genre resource.
frd string <optional>
 false Query Parameter. Specifying frd=true will delete the genre from the database. Otherwise, frd defaults to false; this will flag the genre as removed in the database.
