(static) DELETE Media(:id, frdopt)

Flags a media record as removed or deletes a media record from the database based upon the frd parameter.

Parameters:
Name Type Attributes Default Description
:id ObjectID Route Parameter
frd boolean <optional>
 false Query Parameter. Specifying frd=true will delete the media from the database. Otherwise, frd defaults to false; this will flag the media as removed in the database.
